Adaptive landing page for the movie series "Stranger Things"

## Technologies used

--React, Vite
--Material UI
--Docker, Docker Compose
--Nginx

## Downloading instruction

1.Clone repository
```bash
git clone https://github.com/ananasik5639/devcon.git 
cd devcon
```

2.Download dependencies
```bash
npm install
```

3.Run
```bush
npm run dev
```
4.Run with docker
```bash
docker-compose up --build
```
